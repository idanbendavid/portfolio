When I first thought about this project two things came to mind:
First, I wanted to redeem the grade I received in the Vacations Project and improve my abilities in React and Redux. Second, from a personal point of view, whenever I've tried to exercise, I wasn't able to stay focused thanks to the fact that I wasn't organized enough and didn't have much knowledge in this particular area.


To start the project, I researched an exercise database that would be clear to the user and would be diverse in terms of exercise options, that's how I found the Rapid API exercises database. To build this project using React and Redux, I had to re-learn the basics by creating organized components based on responsibilities and managing the state of various data using Redux-Toolkit which helped me understand how to manage data correctly.


Furthermore, the project design process encouraged me to improve my knowledge of techniques that were not strong such as file uploading from client to server.
Also, I wanted to discover more advanced features like the node-mailer library for node.js to send emails to users.


Has this project given solutions to my thoughts?
Yes, however my job is not done. My next steps are to refactor the code, improve UI visualization and find a suitable hoisting service for me to share the project with the world.
