In all honesty, the topic of this project is not my favorite because eCommerce websites are used in a broad aspect to give answers to many businesses. Despite that, it leads people who used online tools to create those websites to act like they are programmers even though they didn't write code, and to me, coding is work that never ends. There are always tests to do, problems to fix, versions to publish, and features to discover.


Our goal in this project was to practice angular by creating an online grocery store that will provide the ability to buy products and get them delivered to the user's address after completing payment on the site. Implemented features in the project:

  • Subject, Behavior Subject from the rxjs library
  • Pipes
  • Authentication Interceptor
  • Services
  • Guard


    • This project was our 4th and last project in the course, and it granted me the highest score between the exams and projects that took place. The reason that I earned a high score was the fact that I took an active approach by leading and sharing knowledge about specific requirements in the project. That fact allowed me to be among the first to submit the project doing so with confidence that my code is at the highest level.
